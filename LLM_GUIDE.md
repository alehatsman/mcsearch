## Overview

The repository implements a local semantic code-search service named `dex` that indexes Go projects, extracts call graphs, and stores data in an SQLite database using multiple pipelines. The `cmd/dex` package provides command-line tools for managing indices and configurations, while the `internal/store` package handles database operations and query management. The system supports hybrid retrieval methods like cosine similarity and BM25 ranking, and it includes features such as live updates through file watching, static analysis with `golangci-lint`, and integration with OpenAI-compatible endpoints for embeddings and chat completions.
