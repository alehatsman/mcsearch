## Module: scripts

**scripts**

The `scripts` package contains several Bash scripts designed for various tasks related to Go projects, including code quality checks, architecture snapshots, and pre-commit hooks.

- **ai-lint.sh**: Scans Go source files for patterns left by AI agents, outputting machine-readable findings.
- **arch-snapshot.sh**: Generates an architecture snapshot for a Go project, producing a markdown file with package metrics.
- **code-quality-checks.sh**: Checks the current state of code quality soft caps for a Go project, using ANSI escape codes to colorize output based on status.
- **dex-guide-refresh.sh**: Refreshes the index and re-renders `LLM_GUIDE.txt` to ensure it reflects the current state of the codebase.
- **git-hooks-setup.sh**: Sets up Git hooks for a dex development repository, including pre-commit and pre-push hooks that run various checks and analyses.

Dependencies include `gocyclo`, `goda`, `dupl`, and `dex`.
