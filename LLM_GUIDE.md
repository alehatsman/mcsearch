## Overview

This repository contains a suite of Go packages that form an AI-driven code summarization system, designed to understand and summarize the structure and content of software projects. Key components include the `internal/mcp` package for handling symbol queries and graph-based semantic search, the `internal/store` package for managing a database of code summaries, and the `cmd/dex` package for command-line tools that interact with project files and dependencies. The system leverages embeddings for semantic understanding, interacts with an SQLite database through the `store` package, and utilizes various internal packages such as `internal/chat`, `internal/embed`, and `internal/graph` for specific functionalities like graph operations and environment variable resolution. Overall, the repository provides a comprehensive solution for local semantic code-search and summarization, enabling developers to quickly understand and navigate large codebases.
